# Plugins

Plugins are a way to extend the functionality of your community like adding new features, integrations, or customizing the existing ones.

<Callout type="warning" emoji="⚠️">
  Before move on, make sure you have [developer mode](/dev) enabled.
</Callout>

## Create a plugin

Go to AdminCP and go to `Plugins` section. Click on `Create` button and fill the form.

<Img src={createPluginImg} alt="Create a plugin" />

## Developer Tools

Open `AdminCP` and go to `Plugins` page. Choose the plugin you want and click on `Developer Tools` tab.

<Img src={openDevtoolsImg} alt="Open Developer Tool for Plugin" />
