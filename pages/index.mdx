## üé∏ Main Features

- **Open Source**: Core and Forum Plugin VitNode is open source and free to use,
- **Multi-languages Content**: Create content in multiple languages and translate them easily,
- **Themes System**: Customize the look and feel of the website with Tailwind CSS and Shadcn UI,
- **Plugins System**: Extend the functionality of the platform with plugins,
- **Community System**: Create a community around your website with the forum plugin,
- **SEO Friendly**: Optimize your website for search engines with built-in SEO tools _(Enable soon)_,
- **Developer Friendly**: Built with modern technologies and best practices

## Goal of VitNode

The goal of VitNode is to provide a flexible and scalable platform for creating community-driven websites and applications. It is designed to be easy to use and developer-friendly, with a focus on performance and extensibility thanks to [NextJS](https://nextjs.org/) and [NestJS](https://nestjs.com/).
